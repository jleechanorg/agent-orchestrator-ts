# /auton — Autonomy Diagnostic

Ask: **"Why isn't ao autonomously bringing PRs to green?"**

## What to do

1. Read the skill: `.claude/skills/auton.md` in this repo
2. Read the authoritative docs (listed in the skill)
3. Run the diagnostic checklist
4. Output findings in the structured format

The goal is full autonomy: **idea in → merged PR out, zero human clicks.**
If that's not happening, something in the chain is broken — find it.
