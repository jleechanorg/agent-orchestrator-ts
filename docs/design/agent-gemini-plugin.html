Agent Gemini Plugin

Design reference for @composio/ao-plugin-agent-gemini — integrates Gemini CLI into the AO plugin system as a first-class agent backend.

process: gemini session path: ~/.gemini/projects/ runtime: tmux permission flag: --yolo

Purpose

Wraps the gemini CLI so AO can spawn, supervise, resume, and introspect Gemini CLI sessions the same way it manages Claude Code and Codex sessions. No AO core changes required — the plugin implements the standard Agent interface from @composio/ao-core.

Relationship to agent-claude-code

Both plugins are thin wrappers around @composio/ao-plugin-agent-base, which provides the full Agent implementation via createAgentPlugin(config). The two differ only in config:

  • Process name: gemini vs claude
  • Session directory: ~/.gemini/projects/ vs ~/.claude/projects/
  • Permission flag: --yolo vs --dangerously-skip-permissions
  • No --append-system-prompt support in Gemini CLI
  • Cost rates: $0.10/$0.40 per M tokens vs $3/$15 (Gemini Flash baseline)

Shared implementation lives in packages/plugins/agent-base.

Launch

  • gemini — base command
  • --resume <sessionId> — resume existing session (via JSONL UUID)
  • --yolo — auto-approval / permissionless mode
  • --model <name> — optional model override
  • Prompt delivered post-launch via sendMessage()
  • System prompts also delivered post-launch (no CLI flag support)

Session Tracking

  • JSONL files in ~/.gemini/projects/<encoded-path>/
  • Latest file selected by mtime
  • Tail-parse for summary, cost, and session ID
  • toGeminiProjectPath() encodes workspace path identically to how Gemini CLI does

Activity Detection

  • Idle: no JSONL writes in last 8 s
  • Working: recent JSONL write activity
  • Stuck: no new JSONL writes beyond the stuck threshold
  • Process check via ps across all tmux pane TTYs

Agent Interface Implementation

  • getLaunchCommand(config) — builds the gemini CLI string
  • getEnvironment(config) — injects AO_SESSION_ID and metadata env vars
  • isProcessRunning(handle) — checks tmux pane TTYs via ps for gemini
  • getActivityState(handle) — JSONL mtime-based idle/working/stuck classification
  • getSessionInfo(session) — extracts summary, cost, and session UUID from JSONL tail
  • getWorkspaceHooks() — installs metadata-updater hook

Plugin Registration

Registered in the core built-in plugin registry:

// packages/core/src/plugin-registry.ts
{ slot: "agent", name: "gemini",
  pkg: "@composio/ao-plugin-agent-gemini" },

Consumers can also override it per-project via ao.config.json agentPlugin key.

Key Differences from Claude Code

  • Permission flag: Gemini uses --yolo for auto-approval; both permissionless and auto-edit modes map to this flag.
  • No system prompt flag: Gemini CLI has no --append-system-prompt equivalent. System context and prompts are both delivered post-launch via sendMessage().
  • Cost estimation: Falls back to Gemini 2.0 Flash pricing ($0.10/M input, $0.40/M output) when no direct costUSD field is present in the JSONL.

Constraints & Known Gaps

  • Interactive mode by design: AO never passes -p / --prompt — that would cause one-shot exit. The agent is always launched without a prompt, then driven via sendMessage() (promptDelivery: "post-launch"). This is intentional, not a gap.
  • Cost estimation accuracy: Gemini 2.0 Flash pricing ($0.10/M input, $0.40/M output) is used as a baseline. Multi-modal inputs and premium models will be mis-estimated until Gemini CLI exposes per-call cost in JSONL.
  • No system prompt at launch: Gemini CLI has no --append-system-prompt equivalent. System context must be woven into the initial sendMessage() call post-launch.

Testing

  • src/index.test.ts — manifest, launch command, process detection, session parsing (67 tests)
  • src/__tests__/activity-detection.test.ts — JSONL mtime-based activity states with real temp directories (42 tests)
  • All 109 tests pass; no mocks of the filesystem in activity tests (real fs operations)

Future Work

  • Investigate --append-system-prompt equivalent when Gemini CLI adds system prompt support
