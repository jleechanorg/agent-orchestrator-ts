# [agento] fix(launchd): remove KeepAlive from lifecycle-all plist to prevent thrashing

## Overview

- **Root cause**:  on  caused launchd to immediately restart the oneshot  wrapper after every successful exit, exhausting launchd's ThrashInterval protection and deregistering the service
- **Fix**: Removed  from the plist template (and deployed plist). Workers self-manage via pid/lock files — launchd supervision of the wrapper is unnecessary
- **Also**: Reduced  from 60 to 30 per spec
