# [agento] feat(skeptic): switch from ANTHROPIC_API_KEY to llm-eval.ts

## Overview

The skeptic gate had an early-exit check: if `ANTHROPIC_API_KEY` was not configured, it immediately returned `VERDICT: SKIPPED` — before the skeptic CLI even ran. This blocked any repo without `ANTHROPIC_API_KEY` (e.g. jleechanclaw) from using skeptic.
