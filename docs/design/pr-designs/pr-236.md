# [agento] fix(gate-5): check isResolved via GraphQL and exclude PR-author comments

## Overview

Fix gate 5 (inline comments resolved) in both the skeptic-cron.yml 7-green check and merge-gate.ts to correctly identify unresolved blocking comments. Gate 5 detection is precise when GitHub API calls succeed; a known edge case exists when `gh api graphql` silently fails (e.g., when the query argument is omitted) and the workflow falls back to `echo 0`, treating all threads as resolved — the skeptic-cron.yml and merge-gate.ts fixes address both the detection logic and the wire-up of PR author data to avoid false negatives.
