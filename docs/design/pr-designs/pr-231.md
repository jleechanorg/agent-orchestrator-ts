# [agento] feat: generate design docs on PR open + skeptic design-alignment check

## Overview

This change hardens the core lifecycle polling manager around the edges that were still weak after the earlier iterations. The lifecycle loop is the component responsible for normalizing session state, running reactions, and driving operator-visible status transitions, so ambiguous runtime failures here are costly: they make debugging configuration problems harder and can cause otherwise-fixable operator input mistakes to look like missing sessions or silent no-ops.
