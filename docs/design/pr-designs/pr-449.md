# [agento] feat(skeptic): strict video evidence enforcement for UI/terminal claims

## Overview

Evidence standards in this fork mandate that all UI and interactive terminal claims must be backed by video evidence properly captioned and tied to a commit SHA. The `ui-video-evidence.md` and `tmux-video-evidence.md` skills define this policy, but enforcement was missing in the Skeptic evaluator, global config files, and the PR template.
