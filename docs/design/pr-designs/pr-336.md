# [agento] feat: MiniMax agent plugin

## Overview

Add a built-in **agent** plugin so `ao spawn … --agent minimax` runs **Claude Code** against MiniMax’s **Anthropic-compatible** HTTP API (`ANTHROPIC_BASE_URL`, API key via `MINIMAX_API_KEY`), without ad-hoc shell scripts.

**PR:** [#336](https://github.com/jleechanorg/agent-orchestrator/pull/336)

---

## Goals

1. Register `minimax` in the core plugin registry (`BUILTIN_PLUGINS`).
2. Ensure the CLI workspace resolves `@jleechanorg/ao-plugin-agent-minimax` (dependency + `getAgentByName` entry).
3. **Do not** pass Anthropic model IDs on the CLI: strip `--model` on launch; set MiniMax model names via `ANTHROPIC_MODEL` / `ANTHROPIC_SMALL_FAST_MODEL` from `launchConfig.model` or defaults.
4. Disable JSONL **restore** command generation (`getRestoreCommand` → `null`) so resume does not inject incompatible `--model`.
