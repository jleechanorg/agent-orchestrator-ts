# [agento] fix: dedup send-to-agent reactions by PR head SHA (bd-1178)

## Overview

The `send-to-agent` reaction for `changes-requested` fires on every poll cycle even when the PR head SHA hasn't changed. This sends the same review feedback 5-9 times, burning worker context on each duplicate dispatch.
