# [agento] [antig] fix: make worktree path comparison symlink-safe and protect main project root

## Overview

Due to a path comparison mismatch caused by resolved vs unresolved symlinks (e.g. `/var/folders/` vs `/private/var/folders/` on macOS), the safety check in `pruneStaleWorktrees` failed to identify the path as the main project directory when a killed session metadata referenced the project path as its worktree.
