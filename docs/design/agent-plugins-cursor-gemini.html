Cursor & Gemini CLI Agent Plugins

PR: #511 Status: MERGEABLE Date: March 2026
Summary: Added support for Cursor Agent CLI and Gemini CLI as new agent plugins via a shared agent-base factory. This eliminates code duplication and provides unified JSONL/session introspection for all JSONL-based agents.

3. Shared Functionality (agent-base)

  • createAgentPlugin(config) - Factory function
  • toAgentProjectPath() - Path encoding for session dirs
  • setupHookInWorkspace() - Claude Code settings.json hooks
  • parseJsonlFileTail() - Read last N lines from session files
  • extractSummary() - Parse conversation summary
  • extractCost() - Calculate session cost from JSONL
  • getActivityState() - Detect idle/active/exited
  • isProcessRunning() - Check if agent process is alive
